# 🥻 Saree Drapes by Pratibha Jain

A professional, high-end business portfolio website built for a Mumbai-based saree draping artist. This project showcases the intersection of traditional Indian artistry and modern web development.

## 🔗 Live Demo
[View the Site Here](https://aagyajain-eng.github.io/web/) 

---

## ✨ Features
- **Royal Boutique Aesthetic:** A custom-designed theme using a Royal Maroon and Gold "Zari" color palette.
- **Saree Border Design:** Unique CSS-generated borders that mimic the look of premium Banarasi and Paithani sarees.
- **Responsive Layout:** Fully optimized for mobile devices, ensuring clients can book services on the go.
- **Direct WhatsApp Integration:** A "Book Appointment" feature that opens a direct chat with the artist.
- **Meet the Artist Section:** A personal touch to build trust with bridal and party-wear clients.
- **Social Media Connectivity:** Integrated links to Instagram and Google Review systems.

---

## 🛠️ Tech Stack
- **HTML5:** Semantic structure for better SEO and accessibility.
- **CSS3:** Custom styling featuring:
  - Flexbox for responsive layouts.
  - Repeating Linear Gradients for saree border textures.
  - Media Queries for mobile optimization.
- **GitHub Pages:** Used for professional hosting and deployment.
